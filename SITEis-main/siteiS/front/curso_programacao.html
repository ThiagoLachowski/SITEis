▶ Playlist completa

Descrição

O curso de Programação introduz os fundamentos da lógica computacional e ensina a criar seus primeiros programas de forma prática. Você vai aprender a:

  • Compreender conceitos básicos de algoritmos e lógica de programação;
  • Utilizar variáveis, operadores e estruturas condicionais;
  • Criar laços de repetição para automatizar tarefas;
  • Trabalhar com funções e modularizar código;
  • Manipular dados de entrada e saída;
  • Desenvolver pequenos projetos para fixar os conteúdos.

Indicado para iniciantes que desejam dar os primeiros passos no mundo da programação e construir uma base sólida para aprender qualquer linguagem no futuro.
