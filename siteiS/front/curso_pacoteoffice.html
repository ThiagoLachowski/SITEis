Aprendizagem Básica de Informática

▶ Playlist completa

Descrição

Este curso de Aprendizagem Básica de Informática apresenta os conceitos fundamentais para utilização do pacote Microsoft Office e das ferramentas essenciais para o dia a dia: Word para edição de textos, PowerPoint para apresentações, Excel para planilhas e Access para bancos de dados simples. Ideal para iniciantes que desejam ganhar confiança e produtividade em tarefas comuns no computador.

Conteúdo previsto:

  • Introdução ao sistema operacional e navegação por pastas
  • Uso básico do Microsoft Word (formatar textos, salvar, imprimir)
  • Criação de slides no PowerPoint
  • Operações essenciais no Excel (fórmulas simples, formatação, filtros)
  • Conceitos básicos de bancos de dados com Access
  • E mais!
